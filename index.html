About Me

Minha jornada, experiência pelo desenvolvimento de backend

Sou desenvolvedor com foco em Back-End, especializado em APIs escaláveis e microsserviços. Tenho experiência em Java (Spring Boot), Python e TypeScript / JavaScript com Node.js, utilizando frameworks como o Express e o Fastify para o desenvolvimento de APIs em aplicações back-end. Gosto de projetar sistemas seguros e performáticos, usando ferramentas modernas como Docker.

Atualmente curso Ciência da Computação na Universidade Tiradentes e estou sempre em busca de aprimorar minhas técnicas e habilidades para contribuir com soluções criativas. Aplico boas práticas de desenvolvimento e metodologias ágeis como Scrum e Kanban para garantir eficiência e qualidade no processo de desenvolvimento.

Quando não estou programando, gosto de explorar otimização de banco de dados, segurança de aplicações e tecnologias emergentes, sempre buscando construir sistemas preparados para o futuro.

Sistema de Controle de Tarefas

Este é um projeto simples desenvolvido em Java puro (sem frameworks) que simula um sistema de tarefas com persistência em arquivo. É executado totalmente no terminal (modo CLI) e permite criar, listar, editar, concluir e excluir tarefas, além de salvar automaticamente ao sair e carregar ao iniciar.
